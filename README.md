# Broken Tracker

## 🇵🇱 Polski

Broken Tracker to aplikacja działająca jako nakładka (overlay) wyświetlana nad grą. Przechwytuje ruch sieciowy gry i na bieżąco aktualizuje statystyki, nie wymagając ręcznego wprowadzania danych.

## 📦 Instalacja

### 🪟 Windows

1. Pobierz i uruchom plik instalacyjny z rozszerzeniem `.msi`.
2. Jeśli Windows wyświetli ostrzeżenie:
   - Kliknij **„Więcej informacji”**
   - Następnie **„Uruchom mimo to”**
3. Postępuj zgodnie z kreatorem instalacji.
4. Po zakończeniu instalacji uruchom aplikację ze skrótu na pulpicie lub z menu Start.
5. Przy pierwszej instalacji wymagane jest zainstalowanie narzędzia do przechwytywania ruchu sieciowego [](https://npcap.com/#download).
   Zainstaluj je z **domyślnymi ustawieniami**.
---

---

### 🍎 macOS

1. Pobierz plik instalacyjny `.dmg`.
2. Otwórz plik `.dmg` i przeciągnij aplikację do folderu Aplikacje (Applications) lub innej lokalizacji.
3. Uruchom aplikację.
4. Otwórz ** Apple Menu → Ustawienia systemowe**.
5. Przejdź do sekcji **Prywatność i bezpieczeństwo** w panelu bocznym.
6. W sekcji **Bezpieczeństwo** znajdź komunikat o zablokowanej aplikacji i kliknij **„Otwórz mimo to”**.
7. Kliknij ponownie **„Otwórz”**.
8. Przy pierwszym uruchomieniu może pojawić się komunikat o wymaganym dostępie do ruchu sieciowego.
   W takim przypadku pobierz i zainstaluj `BrokenTrackerMacSetup.pkg`.
   Umożliwia on przechwytywanie ruchu sieciowego gry bez konieczności uruchamiania aplikacji jako administrator.

---

### 🤖 Android

1. Pobierz plik instalacyjny `.apk`.
2. Jeśli instalacja z nieznanych źródeł jest zablokowana:
   - Przejdź do **Ustawienia → Bezpieczeństwo**
   - Włącz **„Instaluj z nieznanych źródeł”**
3. Otwórz pobrany plik `.apk`.
4. Zainstaluj aplikację i uruchom ją.


## 🖥️ HUD

HUD oferuje cztery widoki:

- **Przegląd** – najważniejsze statystyki sesji.
- **Instancja** – szczegółowe informacje o aktualnej instancji lub próbie loży.

## ✨ Funkcje

### 💎 Cenny Łup

Dodatkowe okno wyświetlane po zdobyciu cennego przedmiotu.

### ⏰ Przypomnienia o eventach dynamicznych

Konfigurowalne powiadomienia z dowolnym wyprzedzeniem (np. 5 lub 15 minut przed rozpoczęciem eventu). Możliwość dodania nieograniczonej liczby przypomnień.

### 📊 Podsumowanie sesji

Zagregowane statystyki dla wybranego okresu.

### 🏰 Podsumowanie instancji

Statystyki instancji, takie jak zysk na godzinę, psycho na godzinę, zysk na instancję oraz statystyki łączne.

### 📈 Statystyki dzienne

Czytelne wykresy prezentujące dzienne statystyki.

### ⚔️ Historia walk

Pełna lista wszystkich walk z możliwością edycji.

### 📅 Historia aktywności

Rejestr pozostałych aktywności, takich jak eventy dynamiczne, próby loży, daily questy i wiele innych.

### 💰 Rynek

Historia cen rynkowych z możliwością edycji. Możliwość ręcznego ustawienia cen pozostałych przedmiotów, takich jak Wzmocnione Raki, Kryształ Czasu i inne.

### 🌍 Eventy dynamiczne

Lista obserwowanych eventów dynamicznych z odliczaniem do ich rozpoczęcia.

## 🔄 Synchronizacja między urządzeniami

Synchronizacja danych przez lokalną sieć Wi-Fi w dwóch trybach:

- **Ręczna** – wybór urządzenia do synchronizacji.
- **Automatyczna** – działa w tle przy rozpoczęciu i zakończeniu sesji oraz cyklicznie w ciągu dnia.

# 🇬🇧 English

Broken Tracker is an in-game overlay that captures the game's network traffic and updates statistics in real time, eliminating the need for manual data entry.

## 📦 Installation

### 🪟 Windows

1. Download and run the installer file with the `.msi` extension.
2. If Windows shows a security warning:
   - Click **“More info”**
   - Then click **“Run anyway”**
3. Follow the installation wizard.
4. After installation, launch the application from the desktop shortcut or Start Menu.
5. On first installation, you need to install a network packet capture driver [](https://npcap.com/#download).
   - Install it using the **default settings**.

---

### 🍎 macOS

1. Download the `.dmg` installer file.
2. Open the `.dmg` file and drag the application to the Applications folder (or another preferred location).
3. Launch the application.
4. Go to ** Apple Menu → System Settings**.
5. Open **Privacy & Security** in the sidebar.
6. In the **Security** section, find the blocked application message and click **“Open Anyway”**.
7. Click **“Open”** again to confirm.
8. On first launch, you may be prompted to grant network access permissions.
   In this case, download and install the following package `BrokenTrackerMacSetup.pkg`
   This package enables network traffic capture without requiring the application to run as administrator.

---

### 🤖 Android

1. Download the `.apk` installation file.
2. If installation from unknown sources is blocked:
   - Go to **Settings → Security**
   - Enable **“Install unknown apps”**
3. Open the downloaded `.apk` file.
4. Install the application and launch it.

## 🖥️ HUD

The HUD provides four views:

- **Overview** – key session statistics.
- **Instance** – detailed information about the current instance or guild trial.
- **Fight** – statistics and details of the current fight.
- **Party** – information about your party members.

## ✨ Features

### 💎 Valuable Loot

Displays an additional window whenever a valuable item is obtained.

### ⏰ Dynamic Event Reminders

Configurable reminders with any lead time (e.g. 5 or 15 minutes before an event starts). You can create an unlimited number of reminders.

### 📊 Session Summary

Aggregated statistics for a selected time period.

### 🏰 Instance Summary

Detailed instance statistics, including profit per hour, Psycho per hour, profit per instance, and overall totals.

### 📈 Daily Statistics

Daily statistics presented as easy-to-read charts.

### ⚔️ Fight History

A complete list of all battles with the ability to edit entries.

### 📅 Activity History

A log of other activities, including dynamic events, guild trials, daily quests, and more.

### 💰 Market

View and edit historical market prices. You can also manually set prices for other items such as Enhanced Crayfish, Time Crystal, and more.

### 🌍 Dynamic Events

Track dynamic events with a countdown timer until they begin.

## 🔄 Device Synchronization

Synchronize data between devices over your local Wi-Fi network in two modes:

- **Manual** – choose which device to synchronize with.
- **Automatic** – synchronizes in the background when a session starts or ends, and periodically throughout the day.

## ⚙️ Settings

- Application language (Polish / English).
- Display prices in Gold or Platinum.
- Check for and install updates.
