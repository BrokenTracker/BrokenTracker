## 📦 Instalacja
👉 **[Pobierz najnowszą wersję](https://github.com/BrokenTracker/BrokenTracker/releases/latest)**

<details>
<summary>🪟 Windows</summary>
1. Pobierz i uruchom plik instalacyjny z rozszerzeniem `.msi`.
2. Jeśli Windows wyświetli ostrzeżenie:
   - Kliknij **„Więcej informacji”**
   - Następnie **„Uruchom mimo to”**
3. Postępuj zgodnie z kreatorem instalacji.
4. Po zakończeniu instalacji uruchom aplikację ze skrótu na pulpicie lub z menu Start.
5. Przy pierwszej instalacji wymagane jest zainstalowanie narzędzia do przechwytywania ruchu sieciowego [](https://npcap.com/#download).
   Zainstaluj je z **domyślnymi ustawieniami**.
</details>

<details>
<summary>🍎 macOS</summary>
1. Pobierz plik instalacyjny `.dmg`.
2. Otwórz plik `.dmg` i przeciągnij aplikację do folderu Aplikacje (Applications) lub innej lokalizacji.
3. Uruchom aplikację.
4. Otwórz ** Apple Menu → Ustawienia systemowe**.
5. Przejdź do sekcji **Prywatność i bezpieczeństwo** w panelu bocznym.
6. W sekcji **Bezpieczeństwo** znajdź komunikat o zablokowanej aplikacji i kliknij **„Otwórz mimo to”**.
7. Kliknij ponownie **„Otwórz”**.
8. Przy pierwszym uruchomieniu może pojawić się komunikat o wymaganym dostępie do ruchu sieciowego.
   W takim przypadku pobierz i zainstaluj `BrokenTrackerMacSetup.pkg`.
   Umożliwia on przechwytywanie ruchu sieciowego gry bez konieczności uruchamiania aplikacji jako administrator.
</details>

<details>
<summary>🤖 Android</summary>
1. Pobierz `.apk`
2. Otwórz pobrany plik
3. Jeśli instalacja z nieznanych źródeł jest zablokowana
   - Zezwól na instalację dla używanej aplikacji (np. Chrome / Pliki) → „Zezwól na instalację z tego źródła”
4. Zainstaluj aplikację i uruchom ją.
</details>

## 📦 Installation
👉 **[Download latest release](https://github.com/BrokenTracker/BrokenTracker/releases/latest)**

<details>
<summary>🪟 Windows</summary>
1. Download and run the installer file with the `.msi` extension.
2. If Windows shows a security warning:
   - Click **“More info”**
   - Then click **“Run anyway”**
3. Follow the installation wizard.
4. After installation, launch the application from the desktop shortcut or Start Menu.
5. On first installation, you need to install a network packet capture driver [](https://npcap.com/#download).
   - Install it using the **default settings**.
</details>

<details>
<summary>🍎 macOS</summary>
1. Download the `.dmg` installer file.
2. Open the `.dmg` file and drag the application to the Applications folder (or another preferred location).
3. Launch the application.
4. Go to ** Apple Menu → System Settings**.
5. Open **Privacy & Security** in the sidebar.
6. In the **Security** section, find the blocked application message and click **“Open Anyway”**.
7. Click **“Open”** again to confirm.
8. On first launch, you may be prompted to grant network access permissions.
   In this case, download and install the following package `BrokenTrackerMacSetup.pkg`
   This package enables network traffic capture without requiring the application to run as administrator.
</details>

<details>
<summary>🤖 Android</summary>

1. Download the `.apk` file  
2. Open the downloaded file  
3. If installation from unknown sources is blocked:
   - Allow installation for the app you are using (e.g. Chrome / Files) → “Allow from this source”
4. Install and launch the application
</details>
